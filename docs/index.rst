TriFusion's tutorials
=====================

TriFusion is a modern GUI and command line application for gathering,
processing and visualizing phylogenomic data. For the moment, this page is
intended to host the tutorials and how-to guides for using TriFusion. The
complete user guide is
`available as a PDF here <https://github.com/ODiogoSilva/TriFusion/blob/master/docs/manual.pdf>`_.
